Site Name

Anavah Foundation Website

This is the name of an NGO which aims to provide the most basic necessities to to vulnerable communities in Uganda.

Site Purpose

The site provides information about the organization and its purposes. It provides immediate information for people to learn about its founders and mission. The website also offers an option for people to submit their information so that they can be considered for voluntary opportunities.

The website also provides a way for people to make donations to the organization
